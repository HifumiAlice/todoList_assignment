### 내배캠 4~6주차 최민수의 TODO 과제 저장소입니다.

-----------------

### 프로젝트소개

이 프로젝트는 내배캠 과제 Todo 게시판의 백엔드 처리를 담당합니다.
Todo 생성, 조회, 수정 및 삭제와 Todo 글의 댓글 생성, 수정, 삭제를 구현했습니다.

---------

### 개발기간
2024-05-06 ~ 2024-05-24 / 3주간

------------

### 개발환경
IDE : IntelliJ

JDK : 21.0.1

Program Language : Kotlin 1.9

FrameWork : Spring-Boot 3.2.5

Database : postgresql
